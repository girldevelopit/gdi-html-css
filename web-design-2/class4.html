CSS Frameworks

A CSS framework is a library of pre-set CSS styles
or classes that we can use to quickly build projects
or use to add components to projects

Common CSS Frameworks

  • Bootstrap
  • Tailwind
  • Skeleton
  • Foundation
  • Bulma
  • And more!

Best CSS Frameworks in 2022 (Dev.to)

Advantages of Frameworks

  • Give us standardized layouts and styles
  • Provide solutions for complex components:
    • Navbars
    • Carousels
    • Modals
  • Built-in responsiveness; accessibility compliance
  • Speed up our development workflow

Caveats of Frameworks

  • Generic appearance
  • Framework CSS can be difficult to override
  • Heavy reliance on non-semantic selector naming
  • Increases codebase size

Bootstrap

  • Popular CSS Framework
  • Based on Flexbox
  • Uses a 12-column grid system
  • Library of common UI components:
    • Navbars, dropdowns, forms, buttons
    • Typography, cards, carousels and more
  • Responsive and mobile-first

Using Bootstrap

  • Download Bootstrap package OR
  • Link to Bootstrap from an external CDN
  • Link Bootstrap stylesheets and JS files in our HTML file
  • Use Bootstrap class names in our markup

Bootstrap Basics

Bootstrap Breakpoints

Bootstrap has six default breakpoints that control responsiveness:

  • xs X-Small (<576px)
  • sm Small (>=576px)
  • md Medium (>=768px)
  • lg Large (>=992px)
  • xl Extra Large (>=1200px)
  • xxl Extra Extra Large (>=1400px)

Bootstrap Grid

  • A container of rows and columns
  • Rows live inside of containers
  • Columns live inside of rows
  • The grid holds up to 12 columns per row

Basic Bootstrap Grid Syntax

A container or parent element is required
for Bootstrap grid layout to work properly

  • Add a class name of container to a parent element
  • Add a child element with the class name of row
  • Inside the row, add a child element with class name of col

Containers

Bootstrap's grid has three container types:

  • .container: the default, a set max-width
  • .container-{breakpoint}: a responsive container
  • .container-fluid: a 100% [full] width container

Bootstrap Styles and Utilities

Bootstrap offers other uses beyond its grid system,
and where the container element is NOT required:

  • Colors
  • Spacing
  • Flexbox
  • Images
  • Borders
  • And more...

Bootstrap Components

Components, especially JavaScript-based ones, are great ways to use Bootstrap and save time. Common components include:

  • Navbars
  • Buttons
  • Accordions
  • Carousels
