CSS Flexbox Recap

Using CSS Flexbox

We declare flexbox on an element (a parent container)
by using the display property and a value of flex 

.parent {
  display: flex;
}

Flexbox Settings

There are two categories of flexbox properties:

  • flex-container properties for the parent element
  • flex-items properties for the children elements

Flex-Container Properties

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
  • gap

Flex Item Properties

Flex-Item Properties

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

order

Sets the order on a flex item

Items in a flex container are sorted by ascending order and then by their code order 

.flex-child {
  order: 2
}

.flex-child {
  order: -1
}

order property(MDN)

flex-grow

Defines how much a flex-item can grow

.flex-child {
  flex-grow: 0; /* default */
}

flex-grow property(MDN)

flex-shrink

Defines how much a flex-item can shrink

.flex-child {
  flex-shrink: 1; /* default */
}

flex-grow property(MDN)

flex-basis

Defines the width of a flex item

.flex-child {
  flex-basis: auto; /* default */
}

.flex-child {
  flex-basis: 25%;
}

.flex-child {
  flex-basis: 100px;
}

flex-basis property(MDN)

flex

A shorthand property for flex-grow, flex-shrink, flex-basis 

.flex-child {
  flex: 1 1 20%;
}

flex property(MDN)

align-self

Aligns a single flex item on the cross axis

.flex-child {
  align-self: stretch /* default */
}

.flex-child {
  align-self: center;
}

Additional Selectors

There are more ways to be more precise in selecting or targeting HTML elements for styling

The Universal Selector

  • The universal selector selects all elements on a page
  • It is most often used for applying the box-sizing property
* {

  box-sizing: border-box;

}

Box-sizing (W3Schools)

Grouping Selectors

  • The selector list groups other selectors and applies the same style(s) to them 
p, h2, h4 {

  color: purple;

}

Chaining Selectors

We can chain selectors to target more specific elements with CSS

<h2 id="product-name">First Product Title</h2>

<a href="url.com" class="btn-large">Click Me!</a>

h2#productname,

a.btn-large {

  color: purple;

}
/* Only add the color purple to the h2 with an id of product-name, 
and to any anchor tag with the class name of btn-large. */

Attribute Selectors

We can define selectors to identify elements by their attributes

<a href="url.com" class="btn-large">Click Me!</a>

<a href="apple.com" class="btn-large">Click Me!</a>

a[href="apple.com"] {

    color:green;

  }
/* Apply the color green to any anchor tag that links 
to apple.com */

CSS Combinators

CSS Combinators

CSS Combinators describe the relationship between selectors. They help make our CSS more specific. There are four types:

  • Descendant combinator
  • Child combinator
  • Adjacent Sibling combinator
  • General Sibling combinator

CSS Combinators (MDN)

Descendant Combinator

  • Targets all (similar) elements that are descendants of a specified element
  • It is characterized by a single space between the elements
div p {

  color: purple;

}

Child Combinator

  • Targets only child elements that are direct children of the specified parent element
  • It is characterized by a closing angle bracket: >
div > p {

  color: purple;

}

Adjacent Sibling Combinator

  • Targets elements that come directly after each other and share the same parent
  • It is characterized by a plus mark: +
h2 + p {

  color: aquamarine;

}

General Sibling Combinator

  • Targets sibling elements that share the same parent, but do not necessarily come after each other
  • It is characterized by the tilde mark: ~
h2 ~ p {

  color: yellowgreen;

}

Pseudo-Classes

Pseudo-Elements

Pseudo-Classes

Pseudo-classes are selectors that select and style elements in a particular state. For example:

  • Styling anchor tags depending on their state
  • Styling an element when a user hovers/mouses over it
  • Styling the first or last or nth type of an element

Pseudo-Classes (MDN)

Pseudo-Classes

Pseudo-classes start with the target selector/element, then a colon :, and the pseudo-class keyword: 

/* Pseudo-Class Syntax */

selector:pseudo-class-keyword {

  property: value;

}

/* Target every first-child p tag */

p:first-child {

  color:yellowgreen;

}

Hover State


h2 {
  color: red;
}

/* Change the color on hover */

h2:hover {

  color:yellowgreen;

}

Anchor Tag States

Anchor tags exists in the five following states.

Note that there's a pseudo-class for each state.

  • a:link - an unvisited link; the default state
  • a:visited - when the link has been visited
  • a:hover - when a mouse hovers over the link
  • a:active - when the mouse is clicked
  • a:focus - when a user uses the tab key to select a link

Ordering Anchor Tag States

Anchor tag states have to be defined in a specific order in CSS.

  • :hover must come after :link and :visited
  • :active must come after :hover

Styling Links (W3Schools)

Type Position

Pseudo-classes can be used to target specific elements in a group.

/* Target every first-child h3 tag */

h3:first-child {

  color:yellowgreen;

}

Pseudo-Elements

Pseudo-Elements

  • Pseudo-elements add specific styling effects to page elements or add "extra" content to elements
  • Four commonly used pseudo-elements are:
    • ::before
    • ::after
    • ::first-line
    • ::first-letter

Pseudo-Element Syntax

The syntax of pseudo-classes and pseudo-elements look the same, with one exception. Can you spot it? 

/* Pseudo-Class Syntax */

selector:pseudo-class-keyword {

  css-property: value;

}

/* Pseudo-Elements Syntax */

selector::pseudo-element-keyword {

  css-property: value;

}
