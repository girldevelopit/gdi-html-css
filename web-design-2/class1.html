The Position Property

Positioning Elements

A typical webpage contains multiple elements that we may want to position in unique ways or position outside of their normal flow

For example, a sticky navigation or a fixed sidebar or overlapping elements

Golde.co

Position

The position property defines how and where an element can move on the page

It has five values and five coordinate properties:

  • Static
  • Fixed
  • Absolute
  • Relative
  • Sticky
  • Top
  • Bottom
  • Left
  • Right
  • Z-Index
Positioning (MDN)

position: static

This is the default position of all elements; the normal document flow

position: fixed

  • An element is fixed to a position relative to the browser window
  • The element is also moved out of the document flow
  • Coordinates can be applied for precise positioning

.box-three {
  position: fixed;
  top: 10px;
  left: 0;
}

position: absolute

  • The element is positioned relative to its parent container
  • The parent container must have a position value of relative, or the element will be positioned relative to the closest parent with a non-static value
  • The element is removed out of the normal document flow
  • Coordinates can be applied for precise positioning

.box-two {
  position: absolute;
  top: 0;
}

position: relative

  • An element is positioned relative to itself
  • It does not move out of the document flow
  • Coordinates can be applied for precise positioning

.box-five {
  position: relative;
  right: 100px;
}

position: sticky

  • The element is fixed and relative to a parent container until the parent container moves out of the viewport
  • Browser support is not at 100%; use with care

.box-one {
  position: sticky;
  top: 10px;
}

Z-Index

  • The z-index property defines the order of how elements overlap each other
  • The default value of an element's z-index is auto or 0
  • An element with a high z-index will stack over elements with lower z-index values
  • Only works on elements that have a non-static position

.box-four {
  position: sticky;
  top: 10px;
}

CSS Flexbox

CSS Flexbox

Flexbox is a robust CSS module that allows us to easily create responsive layout without floats, tables or positioning

A Complete Guide to Flexbox (CSS Tricks)

Using CSS Flexbox

We declare flexbox on an element (a parent container)
by using the display property and a value of flex,
establishing a flex container to which we can now add
flex settings 

.parent {
  display: flex;
}

Flexbox Settings

There are two categories of flexbox properties:

  • flex-container properties for the parent element
  • flex-items properties for the children elements

Flexbox Axes

Flex children or flex-items are laid out in two directions:

  • The main axis
  • The cross axis

Flexbox Axes

Flex-Container Properties

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
  • gap

flex-direction

Determines in what direction to stack flex items

.flex-parent {
  display: flex;
  flex-direction: row; /* default */
}

Additional values for flex-direction

flex-wrap

Determines if flex items should wrap or not

.flex-parent {
  display: flex;
  flex-wrap: wrap;
}

Additional values for flex-wrap

flex-flow

Shorthand for flex-direction and flex-wrap

.flex-parent {
  display: flex;
  flex-flow: row wrap;
}

Additional values for flex-flow

justify-content

Aligns flex items along the main (or horizontal) axis

.flex-parent {
  display: flex;
  justify-content: center;
}

Additional values for justify-content

align-items

Aligns flex items along the cross (or perpendicular) axis

.flex-parent {
  display: flex;
  align-items: stretch;
}

Additional values for align-items

align-content

Controls spacing between flex items when there's extra space
in the cross-axis and items span across multiple rows;
flex-wrap must be enabled 

.flex-parent {
  display: flex;
  flex-wrap: wrap;
  align-content: center;
}

Additional values for align-content

gap

Determines the spacing between flex items,
but not on their outer edges 

.flex-parent {
  display: flex;
  gap: 1rem;
  gap: 1rem 2rem; /* row-gap, column-gap; */
}
