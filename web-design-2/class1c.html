CSS Flexbox

CSS Flexbox

Flexbox is a robust CSS module that allows us to easily create responsive layout without floats, tables or positioning.

Using CSS Flexbox

We declare flexbox on an element (a parent container)
by using the display property and a value of flex,
establishing a flex container to which we can now add
flex settings 

.parent {
  display: flex;
}

Flexbox Settings

There are two categories of flexbox properties:

  • flex-container properties for the parent element
  • flex-items properties for the children elements

Flexbox Axes

Flex children or flex-items are laid out in two directions:

  • The main axis
  • The cross axis

Flex-Container Properties

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
  • gap

flex-direction

Determines in what direction to stack flex items

.flex-parent {
  display: flex;
  flex-direction: row; /* default */
}

Additional values for flex-direction

flex-wrap

Determines if flex items should wrap or not

.flex-parent {
  display: flex;
  flex-wrap: wrap;
}

Additional values for flex-wrap

flex-flow

Shorthand for flex-direction and flex-wrap

.flex-parent {
  display: flex;
  flex-flow: row wrap;
}

Additional values for flex-flow

justify-content

Aligns flex items along the main (or horizontal) axis

.flex-parent {
  display: flex;
  justify-content: center;
}

Additional values for justify-content

align-items

Aligns flex items along the cross (or perpendicular) axis

.flex-parent {
  display: flex;
  align-items: stretch;
}

Additional values for align-items

align-content

Controls spacing between flex items when there's extra space
in the cross-axis and items span across multiple rows;
flex-wrap must be enabled 

.flex-parent {
  display: flex;
  flex-wrap: wrap;
  align-content: center;
}

Additional values for align-content

gap

Determines the spacing between flex items,
but not on their outer edges 

.flex-parent {
  display: flex;
  gap: 1rem;
  gap: 1rem 2rem; /* row-gap, column-gap; */
}

Syntax for gap

Flex-Item Properties

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

order

Sets the order on a flex item; items in a flex container are sorted by ascending order and then by their code order 

.flex-child {
  order: 2
}

.flex-child {
  order: -1
}

order property(MDN)

flex-grow

Defines how much a flex-item can grow

.flex-child {
  flex-grow: 0; /* default */
}

flex-grow property(MDN)

flex-shrink

Defines how much a flex-item can shrink

.flex-child {
  flex-shrink: 1; /* default */
}

flex-grow property(MDN)

flex-basis

Defines the width of a flex item

.flex-child {
  flex-basis: auto; /* default */
}

.flex-child {
  flex-basis: 25%;
}

.flex-child {
  flex-basis: 100px;
}

flex-basis property(MDN)

flex

A shorthand property for flex-grow, flex-shrink, flex-basis 

.flex-child {
  flex: 1 1 20%;
}

flex property(MDN)

align-self

Aligns a single flex item on the cross axis

.flex-child {
  align-self: stretch /* default */
}

.flex-child {
  align-self: center;
}

Additional values for align-self
